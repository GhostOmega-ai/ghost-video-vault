# 👻 Ghost Video Vault

## v0.4 — Cinema Player

- Removed the rotate-screen control
- Large Play/Pause control centred over the video
- Full-width progress bar across the bottom of the video
- Current time and total duration shown above the bar on the far left
- Clean modern Mute and Fullscreen controls on the far right
- Double-tap left/right thirds still skips backward/forward 10 seconds
- Swipe left/right still changes videos without leaving the player
- Ghost Back, Info, Favourite, Move and Delete controls retained
- Video always uses `object-fit: contain`
- Closing the player automatically exits browser fullscreen

Existing videos, albums, favourites, Private pins and covers remain compatible.
