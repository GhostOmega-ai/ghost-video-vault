# 👻 Ghost Video Vault

## v0.3.1 — Rotation & Icon Polish

- Rotate now works as one smooth tap.
- Landscape playback always uses `object-fit: contain` so the whole video remains visible.
- Leaving the player unlocks orientation, exits fullscreen and restores the normal portrait app view.
- Fullscreen and orientation are handled separately while still allowing rotation to complete in one gesture.
- Replaced the emoji mute control with a clean modern white line icon.
- Added matching line icons for rotate and fullscreen.

Existing videos, albums and IndexedDB data remain compatible.
